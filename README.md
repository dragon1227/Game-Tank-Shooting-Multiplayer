# Game Tank Shooting Multiplayer

### Resumo
O jogo consistem pegar as cargas no centro do mapa e levar de volta a base antes da equipe inimiga. O game conta com mecânicas simples de andar, mirar/atirar e coletar itens no chão.

### Desenvolvimento
Este jogo foi desenvolvido por Luis Henrique, na Unity usando a linguagem c-sharp e alguns assets gratuitos.
